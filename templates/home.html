concept

The aim is to store and retrieve intent data internally within the company. So a new voice design can reuse the already existing intents and example utterances without inventing the wheel again.

Register with your working email.
Search intent with key words in Intents. You can add an intent in AddIntents page. Intent name, description, and minimal five example utterances are required. Entity name and values are optional. To edit an intent, push edit botton to modify the data. Only admin is allowed to delete any data, so please contact the admin for deletion.
